# capistrano-scm-tar

A tar strategy for Capistrano 3 to deploy prebuilt tarball.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'capistrano-scm-tar'
```

Then switch the :scm option to :tar in config/deploy.rb:

```ruby
set :scm, :tar
```
